# tasks/

Working notes for in-flight and recently completed work.

## Files

- **`todo.md`** — current plan with checkable items. Updated at the start of any non-trivial task per the CLAUDE.md workflow. Cleared (or archived inline) when the task is done.
- **`lessons.md`** — append-only log of corrections, gotchas, and rules-for-self captured after the user corrects something or after a non-obvious bug is found. Read this at session start.

## When to update

| Event | File |
|---|---|
| Starting a multi-step task | `todo.md` (write the plan) |
| Marking a step done | `todo.md` (check the box) |
| User corrects me | `lessons.md` (write the rule) |
| Found a non-obvious bug worth remembering | `lessons.md` |
| Task complete | `todo.md` (review section) |
